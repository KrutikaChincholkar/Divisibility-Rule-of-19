# Divisibility-Rule-of-19
A number is divisible by 19 if the following process results in a number divisible by 19. Rule 1. Take the last digit of the number. 2. Multiply it by 2. 3. Add this value to the remaining number. 4. Repeat the process until the number becomes small. 5. If the final number is divisible by 19, then the original number is divisible by 19.
